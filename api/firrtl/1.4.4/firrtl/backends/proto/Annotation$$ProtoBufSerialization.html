ProtoBufSerialization 

case class ProtoBufSerialization(circuit: Circuit, suffix: Option[String]) extends NoTargetAnnotation with CustomFileEmission with Product with Serializable

This will cause the enclosed circuit to be serialized to a Protocol Buffer with the provided suffix.
